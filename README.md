# 🦷 EndoFile Risk v1.0

**EndoFile Risk** es una herramienta digital de apoyo clínico y académico diseñada para la práctica de la endodoncia moderna. Su objetivo es estimar el riesgo de fractura y calcular el número de usos restantes de los instrumentos rotatorios mediante el monitoreo de variables críticas en tiempo real.

---

## ⚠️ Validación Técnica y Científica
Este software ha sido calibrado y validado **exclusivamente** para su uso con limas RISING de Fanta-Dental**. 

> [!IMPORTANT]
> Las propiedades físicas, la memoria controlada y la resistencia a la fatiga cíclica varían drásticamente entre diferentes tratamientos térmicos. El uso de esta herramienta con otras aleaciones (NiTi convencional, Gold, Blue, etc.) puede generar cálculos de fatiga inexactos y comprometer la seguridad del procedimiento.

---

## 🚀 Características Principales
- **Algoritmo RISING:** Implementación digital del sistema de gestión de fatiga para instrumentos Fanta-Dental.
- **Cronómetro de Trabajo Real:** Sincronización precisa para medir el tiempo efectivo de instrumentación.
- **Cálculo de Fatiga Multivariable:** Analiza diámetro ISO (desde #10), conicidad, torque, tiempo y grado de curvatura del conducto.
- **Base de Datos Local (IndexedDB):** Los registros se guardan de forma privada en el navegador del dispositivo, sin necesidad de conexión a internet.
- **Exportación de Datos (CSV):** Genera reportes optimizados que incluyen:
  - Tipo de lima (Diámetro/Conicidad).
  - Usos restantes estimados.
  - Índice de fatiga porcentual.

## 📲 Instalación y Uso Clínico (iPad mini 5 / Mac)
La aplicación está diseñada como una **PWA (Progressive Web App)** para una experiencia fluida junto a herramientas como *EndoIQ*.

### En iPad (Recomendado):
1. Abre el link del proyecto en **Safari**.
2. Toca el botón **Compartir** (cuadrado con flecha hacia arriba).
3. Selecciona **"Agregar a inicio"**.
4. **Tip Clínico:** Abre la app y usa la función **Split View** de iPadOS para colocar *EndoFile Risk* al lado de tu software de motor endodóntico.

### En Mac M1:
1. Abre el link en Chrome o Edge.
2. Haz clic en el icono de **Instalar** en la barra de direcciones.
3. Ancla la app al Dock para acceso inmediato.
